# Cloud-Based RAG System for IT Helpdesk Tickets
This project demonstrates the use of Retrieval-Augmented Generation (RAG) to power intelligent, real-time support for IT helpdesk queries. It combines Azure AI Search for real-time information retrieval and DeepSeekâ€™s LLM API to generate accurate, context-aware responses to support requests.

## Getting Started
To run the project locally:
### 1. Clone the repository
```
https://github.com/ItsJenny1/RAG-Cloud-Computing-Project.git
```
### 2. Navigate to frontend folder
```
cd RAG-Cloud-Computing-Project/frontend
```
### 3. Install dependencies
```
npm install
```
### 4. Start the development server
```
npm start
```
### 5. Open the app
Visit http://localhost:5173 in your browser
